Technical Debt Manager

for Java and Kotlin development teams

Resolvt is a tool that helps you to collect, keep track, manage technical debt across your applications, and does encourage your team to take care of the codebase

How it works

01

Integrate Resolvt with your codebase

Create a workspace by connecting Resolvt to your source repositories.

Supported: GitHub, Bitbucket

02

Install an IDE plugin

Resolvt plugin makes technical debt always visible while you are working with the code. There is no need to leave the IDE to browse and report on technical debt.

Supported: Intellij IDEA

03

That's it

Resolvt keeps track of your repositories events and automatically updates the debt records, trying to keep everything up-to-date.

Supported: GitHub, Bitbucket

Supported: Intellij IDEA
